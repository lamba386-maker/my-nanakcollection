# Nanak Collection - Heritage-Driven Fashion

## Deployment

This website is hosted on **GitHub Pages** for free!

### To deploy:

1. Push changes to the `main` branch
2. Go to Settings > Pages in your repository
3. Select `main` as the source branch
4. Your site will be live at: `https://lamba386-maker.github.io/my-nanakcollection`
